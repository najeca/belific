# Belific

A warm, focused productivity app for managing your daily schedule with Pomodoro timer.

## Features

- **📅 Daily Schedules** - Pre-loaded 14-day exam prep schedule
- **✏️ Edit & Delete Events** - Tap any event to edit or remove
- **📝 Notes** - Add notes to any event
- **⏱️ Pomodoro Timer** - Focus sessions with customizable durations
- **📆 Calendar View** - Navigate and view any day
- **📱 PWA** - Install on your phone's home screen

## Quick Start

1. Deploy to GitHub Pages
2. Open `https://YOUR_USERNAME.github.io/Belific/` on your phone
3. Add to Home Screen via Share menu
